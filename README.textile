h1. FilePath Textmate Bundle

Simple TextMate bundle to add a way to copy a path for the current file you're editing. This is a fork of "bgipsy/files-tmbundle":https://github.com/bgipsy/files-tmbundle to add a slash before the copied path.

The default key binding is command-option-p.

h2. Installation

Here's how you can install it for TextMate 2:

<pre>
  cd "~/Library/Application Support/TextMate/Managed/Bundles"
  git clone http://github.com/hulbert/filepath-tmbundle.git FilePath.tmbundle
</pre>
